Hi,

here is an interesting new application using SIOC, via RDFa mailing
list.

--------- Forwarded message --------
De: Hausenblas, Michael
Para: RDFa
Cc: SWD WG
Asunto: [RDFa application] MLE - mailing list archives explorer
Fecha: Wed, 18 Jul 2007 09:01:16 +0200

Dear RDFa folks,

I'm delighted to announce the availability of MLE (pronounce: Melly)
[1]. MLE is the 'mailing list archives explorer', and basically
enhances (W3C) mailing list archives with SIOC metadata, deployed
with RDFa -as a result, SPARQL queries can be executed on top of it.

With MLE it is demonstrated, how existing data sources can be
enriched/augmented with RDF, to finally enter the Semantic Web.

Any feedback welcome!

Cheers,
Michael

BTW: I recently submitted MLE to the ISWC 2007 challenge [2].

[1] http://sw.joanneum.at/mle/xplore.php
[2] http://challenge.semanticweb.org/
